Vibe coding a small phaser.js game for my partner's birthday. Using Claude Opus 4.7. Sprites are a mix of pixellab.ai, piskelapp.com, and gemini's nano banana.

After cloning, start a web server with `npx serve .` and open the link in your browser (can be on the phone too; tested on iOS, it's better if you add it to home screen instead of running in safari).
